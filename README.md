```markdown
# 📊 PDF読み取り精度バトル評価システム v1.0

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
![Version](https://img.shields.io/badge/version-v1.0-green)

**「PDFをAIに読ませたとき、どれだけ正確に理解できてる？」を楽しく真剣に比較するツールです！**

初心者でも簡単に使えて、AIのPDF読み取り能力を「バトル形式」で採点できます。  
人気格闘漫画風の「戦闘力」計算までついて、結果がドラマチックに表示されます⚔️

## 🎯 どんな人にオススメ？

- AI（ChatGPT、Claude、Gemini、Grokなど）でPDFを扱う人
- 「PDFをそのまま渡す」vs「Markdownに変換してから渡す」どっちがいいか知りたい人
- AIの能力を公平に比較したい人
- ただただ楽しいバトル結果が見たい人

**プログラミング知識は一切不要！**  
チャットAIにコピペするだけで使えます。

## 🚀 特徴（何ができるの？）

- **100点満点の詳細採点**
  - 文字の正確さ（誤字・欠落など）
  - 表や見出しの構造がちゃんと再現されているか
  - 文脈（「前述の通り」などのつながり）が保たれているか
  - クイズ10問で実用性をチェック
- **戦闘力計算＆ランク付け**
  - スコアが高いほど戦闘力が爆上がり！（SSSランク「全知神」まであります）
- **複数AI合議制にも対応**
  - 3つのAIで別々に評価して平均を取ることも可能（より公平に）

## 📖 使い方（超初心者向けステップ）

### 準備するもの
1. 評価したいPDFファイル（どんなPDFでもOK）
2. 比較したい2つの方法
   - 方法A：PDFを**Markdownに変換**してからAIに渡す（おすすめツール：ChatGPT、Claude、または無料のPDF→MD変換ツール）
   - 方法B：PDFを**そのまま直接**AIに渡す
3. 使いたいチャットAI（ChatGPT、Claude、Gemini、Grokなど）

### 手順
1. このリポジトリから **`prompt.md`**（または `main_prompt.txt`）をコピー
2. チャットAIを開いて、新しい会話を作成
3. コピーしたプロンプト全体を貼り付ける
4. プロンプト内の指示に従って：
   - 元のPDFをアップロード（または内容を説明）
   - 方法Aと方法Bの結果をAIに教える
   - 自動で採点＆バトル結果が表示されます！

**詳しい例は `example/` フォルダにサンプルを用意しています！**

### ヒント
- 初めての人は、短いPDF（5〜10ページ程度）で試してみてください
- 「Markdown変換」がわからない場合は、ChatGPTに「このPDFをMarkdownに変換して」と頼むだけでOKです
- 複数AIで合議したい場合は、同じプロンプトを別のAIに貼り付けて結果を比べるだけ！
```
